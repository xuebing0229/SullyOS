一次拖入 / 选择多张角色卡 JSON（或整个文件夹），批量检测里面是否带 API 密钥等危险信息。全程在本地浏览器运行，不上传任何数据。
把 多张角色卡 .json（或一个文件夹）拖到这里，或点击 选择文件
检测逻辑：
1. 深度递归扫描整份 JSON——字段名像密钥（apiKey / secret / token / authorization / bearer / password / anonKey …）或字段值像密钥（sk-…、Bearer …、JWT、长十六进制/base64）的一律标红，新增字段也覆盖。
2. 按已知清单核对美化 / 语言 / 运行时状态字段是否存在。
密钥值默认打码，只露头尾几位供比对。点表格里任意一行可展开该卡详情；可按结论筛选、按文件名搜索、导出 CSV。
